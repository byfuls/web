이미지

<img> - 이미지 삽입

src 속성을 이용해 파일 경로를 지정

호두

alt - 이미지를 설명해 주는 대체 텍스트

width, height - 이미지 크기 조정

지정하지 않으면 원본 크기 그대로 화면에 표시

figure, figcaption - 이미지에 설명 글 붙이기

이미지 뿐만아니라 표, 멀티미디어 요소 등 어디에나 설명 글을 함께 붙일 수 있음

설명 글이 필요한 대상은 <figure> 태그로 묶고,

설명 글은 <figcaption> 태그로 묶음

링크 만들기

<a>태그, href 속성 - 링크 만들기

링크를 만드는 <a> 태그는 텍스트와 함께 사용하면, 텍스트 링크

이미지와 함께 사용하면 이미지 링크

                         텍스트
<a> 태그 속성
속성 설명 option value option detail
href 링크한 문서나 사이트의 주소를 입력
target 현재 창 or 새창 _blank 새창 or 새탭
_self 링크가 있는 화면에서 바뀜(default)
_parent (프레임 사용 시) 부모 프레임에 표시
_top (프레임 사용 시) 프레임에서 벗어나 내용을 전체 화면에 표시
download 보여주는게 아니라 다운로드
rel 해당 문서와 관계
hreflang 해당 문서 언어 지정
type 해당 문서 파일 유형
