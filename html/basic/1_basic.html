텍스트를 한 줄로 표시하는 태그

<strong> vs <b> - 굵게 표시하기

경고, 주의 사항처럼 중요한 내용을 강조할 때, <strong> 태그 (화면 낭독기가 강조되었다고 말해줌)

단순히 굵게 표시할 때 <b> 태그

<em(emphasis)> vs <i(italic)> - 이탤릭체로 표시하기

문장에서 흐람상 특정 부분을 강조하고 싶을 때 <em> 태그 (중요한 내용을 담고 있는 것을 표시)

마음 속의 생각이나 꿈, 기술적인 용어, 다른 언어의 관용구 등에는 <i> 태그

<q> - 인용 내용 표시하기

인용한 내용을 표기하기 위한 quote 줄임말

<blockquote> 는 블록 레벨 태그이기에 줄이 바뀌어 나타나고 구별되도록 안으로 들여 써지지만,

<quote> 태그는 인라인 레벨 태그이기에 줄바꿈 없이, 다른 내용과 한줄로 표시, 구별할 수 있도록 인용 내용에 따옴표 표시

목록을 만드는 태그

<ul> vs <li> - 순서 없는 목록 만들기

순서가 필요치 않는 목록(unordered list)만들 때는 <ul> 태그 사용,

<ul> 태그 안에 <li> (list item) 을 사용해 각 항목 표시

(각 항목 앞에 작은 원이나 사각형 같은 불릿이 붙음)
  • Lorem ipsum dolor sit amet, consectetur adipiscing elit.
  • Maecenas finibus velit vitae risus ullamcorper, id aliquet leo malesuada.
  • Pellentesque a ipsum vitae nisl ultrices faucibus eget sit amet tortor.

<ol> vs <li> - 순서 있는 목록 만들기

순서가 필요한 목록(ordered list)만들 때는 <ol> 태그 사용,

<ol> 태그 안에 <li> (list item) 을 사용해 각 항목 표시

각 항목 앞에 순서 의미하는 숫자 등의 표기가 붙혀지고, 이는 바꿀 수 있음
  1. Lorem ipsum dolor sit amet, consectetur adipiscing elit.
  2. Maecenas finibus velit vitae risus ullamcorper, id aliquet leo malesuada.
  3. Pellentesque a ipsum vitae nisl ultrices faucibus eget sit amet tortor.
