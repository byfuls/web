웹 브라우저의 객체
  • 웹 브라우저에서 사용할 수 있는 호스트 객체를 배움
  • 호스트 객체란?
    ECMAScript로 규정된 기본 객체 외에 자바스크립트 실행 환경에 정의된 객체를 말함
  • 자바스크립트 프로그램은 웹 브라우저에 구현된 다양한 호스트 객체를 이용해 CSS로 스타일을 제어하거나 HTML 문서를 제어할 수 있으며
    웹 브라우저의 Window 객체를 제어할 수 있음
  • 또한, 대화형(Interactive) 웹 페이지를 동적(Dynamic)으로 만들 수 있음

이벤트 처리기 등록하기와 타이머

이벤트 처리기

  • 웹 브라우저에서 동작하는 프로그램은 기본적으로 이벤트 주도형 프로그램(event driven program)
  • 이벤트 란?
    버튼을 클릭하는 행위처럼 단말기와 애플리케이션이 처리할 수 있는 동작이나 사건
  • 이벤트 주도형 프로그램 이란?
    이벤트가 발생할 때까지 기다렸다가 이벤트가 발생했을 때 미리 등록해 둔 작업을 수행
  • 이벤트 처리기 란?
    이벤트가 발생 했을 때 실행되는 함수
    함수를 이벤트 처리기로 등록하는 방법
    • 1. HTML 요소의 속성으로 등록하는 방법
    • 2. DOM 요소의 프로퍼티로 등록하는 방법
    • 3. addEventListener 메서드를 사용하는 방법

[이벤트 처리기 등록] 2. DOM 요소의 프로퍼티로 등록

  • DOM(Document Object Model)은 자바스크립트 등의 프로그램이 HTML 요소를 조작할 수 있게 하는 인터페이스

DOM 객체

  • DOM에서는 HTML 문서나 HTML 요소를 가리키는 객체로 자바스크립트를 사용하여 HTML 문서를 조작
    • window : Window 객체, 웹 브라우저 윈도우 하나 or 탭 하나 를 가르킴
    • document : Document 객체, HTML 문서 전체를 가르킴
      HTML 문서에서 HTML 요소 객체를 가져오거나 HTML 요소를 새로 만드는 등 HTML 문서 전반에 걸친 기능 제공
    • 요소 객체 : HTML 문서의 요소를 가리키는 객체

이벤트 처리기 제거

  • 이벤트 처리기가 등록되어 있지 않은 이벤트 처리기 프로퍼티에는 기본적으로 null이 담겨 있음
  • 그래서 이벤트 처리기를 제거할 때는 단순히 null을 대입
  • (ex) button.onclick = null;

타이머

  • 웹 브라우저의 window객체에는 setTimeout과 setInterval 메서드가 있음
  • 이 메서드를 활용하여
    지정된 시간이 흐른 후에 함수를 실행하거나 (=setTimeout)
    일정한 시간 간격에 따라 함수를 반복 실행 할 수 있음 (=setInterval)

지정된 시간이 흐른 후에 함수 실행하기 : setTimeout

  • 일정 시간이 흐른 후에 한번만 호출되는 함수를 등록
  • setTimeout 첫번째 인수는 '실행하고자 하는 함수의 참조' (문자열을 넘길 수도 있음)
    두번째 인수는 '지연 시간'을 밀리초 단위로 지정
  • setTimeout()의 반환값을 clearTimeout()의 인수로 넘겨서 실행하면 함수 실행이 취소됨
  • 지연 시간을 0으로 지정하면 즉시 실행되는 것 같지만 실제로는 실행 중인 이벤트 처리기의 작업이 끝나면 가능한 빨리 실행함

지정된 시간마다 반복해서 실행하기 : setInterval

  • 일정한 시간 간격에 따라 반복해서 실행할 함수를 등록할 수 있음
  • setInterval 첫번째 인수는 '실행하고자 하는 함수의 참조' (문자열을 넘길 수도 있음)
    두번째 인수는 '시간 간격'을 밀리초 단위로 지정
  • setInterval()의 반환값을 clearInterval()의 인수로 넘겨서 실행하면 함수 실행이 취소됨

HTML 요소를 동적으로 읽고 쓰기

  • 자바스크립트를 활용해서 HTML 요소 안의 내용을 동적으로 읽고 쓰는 방법을 설명
  • 이 방법을 배우면 HTML의 폼 요소 등을 사용자 인터페이스로 삼아 프로그램 입출력을 할 수 있게 됨

HTML 요소의 innerHTML 프로퍼티로 읽고 쓰기

  • 요소 객체의 innerHTMl 프로퍼티는 그 HTML 요소의 내용을 가르킴
  • 이로써 HTML 요소의 내용을 읽거나 쓸 수 있음

폼 컨트롤의 입력 값 읽기

  • input 요소 등의 폼 컨트롤 요소를 사용하면 사용자의 입력 값을 자바스크립트 프로그램에서 사용할 수 있음
    <폼 컨트롤 요소의 값을 구하는 프로퍼티>
    요소 type 속성의 값 프로퍼티 설명
    input number, text, 등 value 입력된 값을 문자열로 변환한 값
    checkbox, radio checked 항목의 선택 여부를 뜻하는 논리값
    select - selectedIndex 선택된 option 요소를 가리키는 0부터 시작하는 번호
    textarea - value 입력된 문자열

document.write

  • 초창기 자바스크립트에서는 document.write가 처리 결과를 출력하는 유일한 수단
  • 현재는 DOM 상요해서 HTML 요소를 생성하거나 처리 결과를 HTML 문서로 출력하는 것이 일반적
  • 따라서 document.write 사용하는 경우는 매우 드뭄
  • 그러나 자바스크립트를 배울 때는 간편한 출력 수단으로 활용할 수 있으므로 가끔 사용
  • document.write 메서드는 인수로 받은 문자열을 HTML 문서의 body 요소 안에 출력
    이렇게 되면 HTML 문서의 내용이 바뀌게 되고 웹 브라우저는 수정된 HTML 문서를 표시
  • 웹 브라우저가 HTML 문서를 해석하는 도중에 document.write ㄹㄹ 실행한다는 점에 유의
  • 동적으로 HTML문서를 변경하려면 DOM을 사용하는것을 권장

Canvas를 활용한 컴퓨터 그래픽스

  • Canvas는 웹 브라우저에서 그래픽을 처리하기 위해 추가된 HTML5 구성 요소
  • 원래는 웹 브라우저에서 사용자 인터페이스를 만들기 위해 위젯이나 이미지를 렌더링할 목적으로 개발한 것
  • Canvas는 주요 웹 브라우저에서 사용할 수 있으며 인터넷 익스플로러에서는 IE9 버전부터 사용 가능
  • Canvas는 2차원 그래픽과 WebGL을 사용한 3차원 그래픽을 구현할 수 있음

Canvas의 특징

  • Canvas는 즉시 실행형 저수준 api
  • 저수준
    Canvas는 선, 원, 사각형을 그리는 등의 기본적인 그리기 기능한 제공하므로 그리는 속도가 빠름
    즉 저수준 API란 기본적인 기능만 제공하는 API라는 뜻
    따라서 더욱 복잡한 그림은 저수준 API를 활용한 애플리케이션으로 그림
  • 즉시 실행형
    Canvas의 그리기 명령은 호출하는 즉시 실행됨
    그림 상태를 저장하는 중간 데이터 계층이 없음

Canvas 좌표계

  • Canvas 좌표계는 기본적으로 왼쪽 윗부분이 원점 (0, 0)
  • 캔버스 오른쪽이 x축의 정방향, 캔버스 아래쪽이 y축의 정방향 (좌표축의 길이를 세는 단위는 픽셀, 기본 좌표계는 나중에 바꿀 수 있음)

사각형 그리기

  • Canvas에 내장된 그리기 기능은 사각형 그리기 하나뿐, 나머지 도형은 나중에 언급하는 패스로 정의해서 그림
  • 사각형을 그리는 방법은 세 가지, 이 방법 모두 사각형 왼쪽 위 끝점 좌표인 (x, y), 사각형 너비인 width, 사각형 높이인 height를 인수로 지정해서 그림
    1. 사각형 테두리 그리기 : strokeRect(x, y, width, height)
      : 좌표(10, 10)에 너비가 200이고 높이가 150인 사각형 테두리를 그린다
      = ctx.strokeRect(10, 10, 200, 150);
    2. 사각형 채우기 : fillRect(x, y, width, height)
      : 좌표(50, 40)에 너비가 120이고 높이가 90인 사각형을 채운다
      = ctx.fillRect(50, 40, 120, 90);
    3. 사각형 영역을 지우고 투명하게 만들기 : clearRect(x, y, width, height)
      : 좌표(90, 65)에 너비가 40이고 높이가 40인 사각형 영역을 삭제한다
      = ctx.clearRect(90, 65, 40, 40);

패스로 그리기

  • 패스를 정의하면 곡선을 그릴 수도 있고 내부를 채울 수도 있음
