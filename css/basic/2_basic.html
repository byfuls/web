font-family 속성 - 글꼴 지정하기

  • font-family 속성으로 지정
  • <body>, <p>, <hn> 태그들 처럼 텍스트를 사용하는 요소들에 주로 사용
  • 한 가지 글꼴만 지정하기도 하지만, 지정한 글꼴이 없을 경우에 대비해 두 번째, 세 번째 글꼴까지 지정
    두 개 이상의 글꼴 이름 지정시 쉼표(,)로 구분
    body { font-family: "맑은 고딕", 돋움, 굴림 }
    맑은 고딕이 없으면 돋움, 돋움이 없으면 굴림

@font-face 속성 - 웹 폰트 사용하기

  • 웹 폰트란 웹 문서를 작성할 때 웹 문서 안에 글꼴 정보도 함께 저장했다가, 사용자가 웹 문서에 접속하면 글꼴을 사용자 시스템으로 다운로드시키는 방식
    사용자가 글꼴이 없더라도 웹 문서를 통해 필요한 글꼴을 다운로드한 후 표시하기 때문에 웹 제작자가 의도한 대로 텍스트를 표시할 수 있음

  • https://fonts.google.com에 접속해서 구글에서 제공하는 웹 폰트 사용 가능

font-size 속성 - 글자 크기 조절하기

px 단위 사용하기

  • 웹에서 font-size 속성을 사용해 폰트 크기 지정할 떄는 주로 px단위를 많이 사용
  • 폰트 크기가 고정되기 때문에 창 크기가 작은 모바일 기기로 볼때도 같은 크기로 화면에 표시. 결국 작은 화면 안에 작은 글씨로 표시
    모바일 기기까지 고려한다면 px단위보다 em단위 사용이 좋음
    (크기 따로 지정하지 않으면 기본 16px)

em 단위 사용하기

  • 부모 요소에서 지정한 폰트의 대문자 M의 너비를 1em으로 놓고 상대적 값을 계산해 다른 요소들의 글자 크기를 조절
  • 만약 지정한 크기가 없다면 <body>요소의 크기 16px이 기본값 1em으로 지정

텍스트 스타일

  • 글꼴은 폰트와 관련된 내용
  • 텍스트 스타일은 글자와 단어들, 글자들로 이루어진 문단에서 사용되는 스타일

color 속성 - 글자 색 지정하기

  • 문단이나 제목 등의 텍스트에서 사용되는 글자 색을 바꿀 때는 color 속성을 사용
  • 색상 값은 16진수나 rgb(or rgba), hsl(or hsla) 또는 색상 이름으로 표기

text-decoration 속성 - 텍스트에 줄 표시하기/없애기

  • 텍스트에 밑줄 표시
  • 텍스트에 취소선 표시

text-transfrom 속성 - 텍스트 대.소문자 변환하기

  • 텍스트를 대.소문자로 변환
  • 텍스트를 전각문자로 변환

text-shadow 속성 - 텍스트에 그림자 효과 추가하기

  • 제목이나 강조해야 할 글자에 사용하면 눈에 띄는 효과
  • 텍스트를 좀 더 입체적으로 표시
  • 그림자 값은 쉼표로 구분해 여러값을 지정 할 수 있음
    보통 그림자 값은 하나만 사용하지만 여러 개를 사용하면 마치 그래픽으로 처리한 듯한 텍스트를 만들 수 있음

white-space 속성 - 공백 처리하기

  • 텍스트와 함꼐 연속해 입력된 여러 개의 공백을 어떻게 처리할지 지정할 수 있음

letter-spacing과 word-spacing 속성 - 텍스트 간격 조절하기

  • 텍스트 자간 조절
  • 강조하고 싶은 글자나 <hn>태그를 이용해 표시한 글자들은 글자 사이 간격을 조절해 좀 더 여유 있게 표시 할 수 있음
  • letter-spacing 속성은 낱 글자 사이 간격을 조절하고
    word-spacing 속성은 단어와 단어 사이 간격을 조절
    (주로 단어 사이 간격은 수정하지 않고, letter-spacing속성을 사용해 자간을 조절)

문단 스타일

  • 문단 스타일에서 조절하는 부분은 "텍스트 정렬", "들여쓰기", "줄 간격" 등

direction 속성 - 글자 쓰기 방향 지정하기

  • 텍스트를 어느 방향부터 쓰기 시작해 화면에 표시할지를 결정

text-align 속성 - 텍스트 정렬하기

  • 문단의 텍스트 정렬 방법 지정
  • "왼쪽 정렬", "오른쪽 정렬", "양쪽 정렬", "가운데 정렬"

text-justify 속성 - 정렬 시 공백 조절하기

  • text-align 속성 값이 justify인 경우, 양쪽 끝에 맞추기 때문에 글자와 단어 사이 간격이 어색하게 벌어질 수 있음
    이때, 간격을 어떻게 조절해 정렬할 것인지 지정하는 것이 text-justify

text-indent 속성 - 텍스트 들여쓰기

  • 문단의 첫 글자를 얼마나 들여 쓸지를 지정

line-height 속성 - 줄 간격 조절하기

  • 원하는 만큼 줄 간격 조절
  • <숫자>나 <백분율>로 지정할 때는 글자 크기를 기준으로 설정
    (ex) 글자 크기가 12px이고 줄 간격을 2.0으로 했다면, 실제 줄 간격은 글자 크기의 2.0배인 24px가 됨 (백분율도 같은 식으로 계산)
    보통 줄 간격은 글자 크기의 1.5 ~ 2배가 적당

text-overflow 속성 - 넘치는 텍스트 표기하기

  • white-space: nowrap으로 지정해 줄 바꿈을 하지 않을 때는 텍스트가 기준선을 벗어나 넘칠 수 있음
    이때, 넘치는 텍스트를 어떻게 처리할지 지정할 수 있음
  • 사용할 수 있는 값은 clip과 ellipsis가 있음
  • 다만, text-overflow속성은 해당 요소에서 overflow 속성값이 hidden이거나 scroll, auto이면서 white-space: nowrap일 경우에만 적용

list-style-type 속성 - 목록의 불릿과 번호 스타일 지정하기

  • 순서 없는 목록의 경우 목록 앞에 불릿 넣을 수 있고
    순서 있는 목록의 경우 목록 앞에 번호 스타일을 넣을 수 있는데
    이 때 사용하는 속성이 list-style-type

순서 없는 목록에서 불릿 모양 바꾸기

순서 목록에서 숫자 바꾸기

list-style-image 속성 - 불릿 대신 이미지 넣기

  • 불릿을 원하는 이미로 바꿈

list-style-position 속성 - 목록에 들여 쓰는 효과 내기

  • 앞서 배운것으로 목록을 만들 경우, 불릿이나 번호는 실제 내용의 바깥쪽에 표시됨
    하지만 list-style-position 속성 사용하면 실제 내용이 시작되는 위치에 불릿이나 번호를 표시하기 때문에
    결과 화면에는 불릿이나 번호가 좀 더 안쪽으로 들여 써진 듯한 효과를 낼 수 있음

list-style 속성 - 목록 속성 한꺼번에 표시하기

  • list-style-type:none; 을 list-style:none;으로 대체해 사용 가능 
  •                   ul {
                    list-style-type: lower-alpha;
                    list-style-position: inside;
                  }
    의 속성값을 아래로 한번에 표시 가능 
                      ul {
                    list-style: lower-alpha, inside;
                  }
