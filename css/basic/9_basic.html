CSS3와 애니메이션

변형

  • 특정 요소의 크기나 형태가 변하는 것을 변형 또는 트랜스폼(transform)
  • CSS를 이용하면 사용자의 동작에 반응해 텍스트나 이미지 등을 움직이게 할 수 있음

2차원 변형과 3차원 변형

  • CSS3이용하면 스타일 시트 소스만으로 간단히 이미지를 변형
  • 이미지의 변형은 크게 '2차원 변형'과 '3차원 변형'으로 나뉨
  • '2차원 변형'은 단순히 수평이나 수직으로 이동하고 회전하는 것을 말함
    x축과 y축으로 이동하는 것이며 x축은 오른쪽으로 갈수록 커지고 y축은 아래로 내려갈수록 커짐
  • '3차원 변형'은 x축과 y축에 원근감을 주는 z축을 추가해 변형
    z축은 앞뒤로 이동하는데 사람이 보는쪽으로 다가올수록 값이 커지고 뒤로 갈수록 값이 작아짐

transform과 변형 함수

  • 이미지를 회전시키거나 이동하는 등 웹 요소를 변형하려면 transform 속성을 사용
  • transform: 다음에 변형 함수를 함께 입력

2차원 변형 함수

  • 2차원 변형 함수는 최신 브라우저에서는 모두 지원되지만
    인터넷 익스플로러 9를 비롯한 이전 브라우저를 위해
    -webkit-, -moz-, -ms-, -o- 등의 브라우저 접두사를 붙여야함

3차원 변형 함수

  • 2차원 변형 함수에 z축을 추가하면 3차원 변형 함수가 됨
  • 3차원 변형 함수도 최신 브라우저에서는 모두 지원되지만
    -webkit-, -moz-, -ms-, -o- 등의 브라우저 접두사를 붙여야함
    인터넷 익스플로러의 경우, 10 이상에서 3차원 변형 함수를 지원하므로 -ms-접두사를 따로 사용하지 않음

translate 변형 함수 - 요소 이동시키기

  • translate함수는 2차원과 3차원에서 모두 가능한 변형 함수
  • x축 방향이나 y축 방향 또는 양쪽 방향으로 이동할 거리를 지정해 해당 요소를 지정한 크기만큼 이동

scale 변형 함수 - 요소 확대/축소하기

  • 지정한 크기만큼 확대/축소
  • 괄호 안의 값이 1보다 크면 확대되고 1보다 작으면 축소됨
  • scale함수에 하나의 값만 지정하면 x,y축 같은 값으로 설정됨
