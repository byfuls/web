CSS3와 애니메이션

트랜지션

  • 앞전 요소에서는 이동, 회전, 왜곡 등 비교적 단순한 변형을 배움
    이번 transition은 변형을 넘어 하나의 스타일에서 다른 스타일로 완전히 바꿈
  • 스타일 바뀌는 시간을 조절하면 애니메이션 효과도 낼 수 있음
    CSS 소스만으로 애니메이션효과를 내는 방법을 알아봄

트랜지션이란

  • 트랜지션(transition)이란? 웹 요소의 배경 색이 바뀌거나 도형의 테두리가 원형으로 바뀌는 것처럼 스타일 속성이 바뀌는 것을 말함
  • 시간에 따라 웹 요소의 스타일 속성이 조금씩 바뀌는 것을 트랜지션이라고 함

transition-property 속성 - 트랜지션을 적용할 속성 지정하기

  • 트랜지션을 만드는 첫번째 단계는 트랜지션을 어느 속성에 적용할 것인지 선택 하는것
    transition-property 속성으로 이것을 결정
  • transition-property 사용하지 않을 경우, 모든 속성이 트랜지션 대상이 되고 특정 속성 이름을 입력하면 그 속성에 트랜지션이 적용됨

transition-duration 속성 - 트랜지션 진행 시간 지정하기

  • transition-property에서 트랜지션 대상을 지정했다면, 진행 시간을 지정해야 그 시간 동안 속성이 자연스럽게 바뀌는 애니메이션 효과를 만들 수 있음
  • transition-duration 트랜지션 진행 시간이 0초를 기본으로 설정, 단위는 초(seconds) 또는 밀리초(milliseconds)
  • 대상이 되는 속성이 여러 개라면 트랜지션 진행 시간도 쉼표로 구분해 순석대로 여러 개를 지정할 수 있음

transition 속성 - 트랜지션 속성 한꺼번에 표기하기

  • transition 속성은 값의 나열 순서를 따라야함
    (4개의 속성 중 빠진 것이 있다면 그 속성의 기본 값을 사용)
  • 2개 이상의 변형 동시에 사용가능

애니메이션

  • CSS5의 다른 속성인 animation속성을 이용하면 트랜지션보다 더 쉽게 애니메이션을 만들 수 있음
  • 애니메이션과 트랜지션의 차이를 살펴보고 animation속성을 이용해 애니메이션을 만드는 방법도 알아보겠음

CSS와 애니메이션

  • animation속성을 사용하면 자바스크립트나 플래시를 사용하지 않고도 웹 요소에 애니메이션을 추가 할 수 있음
  • 시작 스타일과 끝 스타일을 지정하면 CSS에서 중간 스타일을 자동으로 추가해 전체적으로 부드럽게 변하는 애니메이션 효과를 만들어 낸다는 것은
    트랜지션과 비슷하지만 CSS 애니메이션은 시작해 끝내는 동안 원하는 곳 어디서든 스타일을 바꾸며 애니메이션을 정의할 수 있다는 점은 트랜지션과 다름
  • 애니메이션 중간에 스타일이 바뀌는 지점을 키프레임(keyframe)이라 함

CSS 애니메이션에서 사용하는 속성

  • 애니메이션 만들 때 애니메이션이 바뀌는 지점(키프레임)은 @keyframes 속성을 이용해 정의하고
    animation 속성과 animation하위 속성을 이용해 애니메이션의 실행 시간이나 반복 여부 등을 지정
  • 이 속성들은 인터넷 익스플로러 10 이상과 최신 모던 브라우저에서 지원
    이전 모던 브라우저를 고려하려면 -webkit-, -moz-, 접두사를 붙여야함

@keyframes 속성 - 애니메이션 지점 설정하기

  • 애니메이션의 시작 지점부터 설정
  • 애니메이션의 시작과 끝을 비롯해 상태가 바뀌는 부분이 있다면 @keyframes속성을 이용해 바뀌는 지점을 설정
  • 스타일 속성 값이 바뀌는 지점은 '선택자'로 입력
  • 애니메이션에서 중간지점을 추가하려면 시작 위치를 0%, 끝 위치를 100%로 놓고 50%위치에 키프레임을 하나 더 추가하면 됨
  • 시작과 끝 위치만 사용하겠다면 0%와 100%값 대신 from과 to라는 키워드를 사용해도 됨

animation-name 속성 - 애니메이션 이름 지정하기

  • CSS 애니메이션을 만들때 @keyframes속성을 이용해 여러 개의 애니메이션을 정의하는데,
    animation-name속성에서 지정한 애니메이션 이름으로 구분
  • @keyframes속성을 사용할 때도 @-webkit-frames나 @-moz-keyframes처럼 브라우저 접두사를 붙여야함
