css 포지셔닝과 주요 속성들

css 포지셔닝

  • css 포지셔닝을 이용해 검색 창, 로그인 창, 광고, 뉴스 실시간 검색 순위 등 여러 요소를 적절히 배치
  • 브라우저 화면 안에 각 콘텐츠 영역을 어떻게 배치할지 결정하는 것이 float속성과 position 속성

box-sizing 속성 - 박스 너비 기준 정하기

  • 웹 문서에서 여러 요소를 배치하려면 각 요소의 너비를 계산해야함

  • 박스 모델의 너비를 정확히 계산해야하는데, 계산하기 어렵다면 box-sizing: border-box; 로 지정해 너비를 알기 쉽게 하는 것도 좋은 방법

float 속성 - 왼쪽이나 오른쪽으로 배치하기

  • float 속성은 웹 요소를 문서 위에 떠있게 만듬
    여기서 떠 있다는 왼쪽 구석이나 오른쪽 구석에 요소가 배치된다는 뜻
  • float에서 사용할 수 있는 값은 왼쪽(left)와 오른쪽(right), 그리고 좌우 어느 쪽도 아닌 것(none) 이 있음
  • 이미지를 왼쪽에 떠 있게끔 float를 사용하면 그 주변을 다른 요소가 감싸게 된다
    따라서, 이미지와 텍스트를 나란히 표시하려고 할 떄는 이미지에 float속성을 사용.
    이미지를 왼쪽에 배치한 후 오른쪽에 오는 텍스트와 적당한 간격을 유지하기 위해 margin-right속성도 함께 사용

clear 속성 - float 속성 해제하기

  • float속성을 이용해서 왼쪽이나 오른쪽 배치하면, 그 다음에 넣는 다른 요소들에도 똑같은 속성이 전달됨
  • 따라서 float속성을 더 이상 유용하지 않다고 알려주는 속성이 필요한데 이것이 clear속성
    float:left 이용해 왼쪽으로 배치했다면, clear:left로 종료하고
    float:right 이용해 오른쪽으로 배치했다면, clear:right로 무효화 시킴
    float속성 값이 left인지 right인지와 상관없이 기본 상태로 되돌리고 싶다면 clear:both 사용

position 속성 - 배치 방법 지정하기

  • position속성은 웹 문서 안의 요소들을 자유자재로 배치해주는 속성으로 html과 css를 이용해 웹 문서를 만들 때 중요하게 사용하는 속성 중 하나
  • position속성을 이용하면 텍스트나 이미지를 나란히 배치할 수 있고, 여러개의 요소를 가로나 세로로 원하는 위치에 배치할 수 있음
  • static을 제외한 나머지 속성 값에서는 좌표를 이용해 각 요소의 위치를 조절 할 수 있음
    이때 위치는, top bottom left right 로 지정
    top이란 위쪽에서 얼마나 떨어져 있는지
    bottom은 아래에서 얼마나 떨어져 있는지
    left는 왼쪽에서 얼마나 떨어져 있는지
    right는 오른쪾에서 얼마나 떨어져 있는지
속성 값 설명
static 요소의 문서를 흐름에 맞추어 배치 (기본값)
relative 이전 요소에 자연스럽게 연결해 배치하되, 위치를 지정할 수 있음
absolute 원하는 위치를 지정해 배치
fixed 지정한 위치에 고정해 배치 (화면에서 요소가 잘릴 수도 있음)

static 속성 값 - 문서의 흐름대로 배치하기

  • position속성의 기본값으로 요소를 나열한 순서대로 배치
  • top,bottom,left,right속성 사용할 수 없음
  • float속성을 이용해 좌우로 배치할 수 있음

relative 속성 값 - 문서 흐름 따라 위치 지정하기

  • position:relative도 static에서처럼 나열한 순서대로 배치되지만,
    top,bottom,right,left 속성을 사용할 수 있으며 좌푯값을 사용해 위치를 지정할 수 있음
  • relative로 설정하고 좌표를 설정하면, 원래 표시되었어야 할 위치를 기준으로 이동해 표시

absolute 속성 값 - 원하는 위치에 배치하기

  • 문서의 흐름과 상관없이 left,right,top,bottom 속성 값을 이용해 요소를 원하는 위치에 배치
  • 이때 기준이되는 위치는 가장 가까운 부모 요소나 조상 요소 중 position속성이 relative인 요소
  • 그래서 absolute를 사용하려면 그 요소를 감싸는 <div>를 만들고 positiondmf relative로 지정해 놓고 사용
    absolute속성 값을 이용해 자유자재로 요소를 배치하기 위해서는 반드시 부모 요소가 relative로 지정되어 있어야 한다

fixed 속성 값 - 브라우저 창 기준으로 배치하기

  • fixed 속성 값도 absolute 속성 값처럼 문서의 흐름과 상관없이 위치를 좌표로 결정하지만, 부모 요소가 아닌 브라우저 창이 기준
  • 브라우저 창의 왼쪾 위 꼭지점을 원점으로 두고 좌표가 계산
  • 한번 배치되면 fixed라는 이름처럼 브라우저 창을 스크롤하더라도 계속 고정되어 표시

visibility 속성 - 요소를 보이게 하거나 보이지 않게 하기

  • 특정 요소를 화면에 보이게 하거나 보이지 않게 또는 겹치게 하는 설정
  • 감춘 요소들은 눈에 보이지는 않지만 실제로는 공간을 차지함

z-index 속성 - 요소 쌓는 순서 정하기

  • 한 요소 위에 다른 요소를 쌓을 수 있음
  • 요소 위에 요소를 쌓을 때 쌓는 순서를 지정하는 것이 z-index 속성
  • z-index값이 작을수록 아래에 쌓이고 z-index값이 클수록 z-index 값이 작은 요소보다 위에 쌓임
  • z-index값을 명시하지 않을 경우, 웹 문서에 맨 먼저 삽입하는 요소가 z-index:1 값을 가지며 그 후 삽입하는 요소들은 z-index값이 점점 커짐

다단으로 편집하기

  • 다단 관련 속성은 브라우저별 접두사를 붙여 사용해야함
    크롬, 사파리 -webkit-
    파이어폭스 -moz-
    오페라 -o-
    익터넷 익스플로러 -ms-

column-width 속성 - 단의 너비 고정하고 다단 구성

  • 단의 너비를 고정해 놓고 화면을 분할 할 수 있고
    단의 개수를 고정해 놓고 화면을 분할 할 수 있음
  • 단의 너비를 고정해 놓고 구성하는 방법은 화면이 커지면 단의 개수가 많아지고
    화면이 좁아지면 단의 개수가 줄어듬
  • 다단 편집시 단의 너비를 고정해 놓으려면 column-width 속성 이용

  • 하지만 영문 내용을 여백 없이 길게 입력할 경우, <td>의 width속성은 무시되고 영문 내용은 모두 한 줄로 표시됨
  • 이때 table-layout속성을 이용하면 셀 안의 내용 양에 따라 셀 너비를 변하게 할지, 고정시킬지를 결정할 수 있음

  • 그런데 table-layout:fixed로 설정해 너비를 고정하면 셀 너비보다 긴 내용은 셀 밖으로 밀려나가 버림
